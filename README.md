<h1>Pokemon API</h1>
<h2>Projeto</h2>
Este projeto é uma API desenvolvida para realização de buscas de informações sobre pokemons.

<h2>Instalação</h2>
<ol>
  <li>Clone este repositório: git clone https://github.com/nathaliab03/pokemonApi.git</li>
  <li>Navegue até o diretório do projeto: cd pokemonApi\backend</li>
  <li>Instale as dependências: npm install express</li>
</ol>

<h2>Uso</h2>
<ol>
  <li>Ainda no diretório do backend inicie o servidor no terminal: npm start </li>
  <li>Agora é só abrir o front-end e se divertir :D</li>
</ol>
